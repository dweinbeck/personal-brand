## v1.1 Requirements

Requirements for v1.1 Page Buildout & Polish. Each maps to roadmap phases.

### Projects Page

- [x] **PROJ-01**: Projects page displays cards in a 2-across grid layout with more detail than home page
- [x] **PROJ-02**: Each project card shows project name and one-paragraph description
- [x] **PROJ-03**: Each project card shows topic/software/platform tags
- [x] **PROJ-04**: Each project card shows date initiated and last commit date
- [x] **PROJ-05**: Each project card shows public or private designation
- [x] **PROJ-06**: Each project card has a large button linking to a project-specific page (placeholder/dead link for now)
- [x] **PROJ-07**: Page has headline "Current and Past Projects" with same background as home page
- [x] **PROJ-08**: User can filter or sort projects by tag or date

### Writing Page

- [x] **WRIT-01**: Writing page displays with title "Writing" and subtitle "Articles and Blog Posts by Dan"
- [x] **WRIT-02**: Article cards display in same format/style as Projects page cards
- [x] **WRIT-03**: Each article card shows article title, publish date, and topic tag
- [x] **WRIT-04**: Page renders with lorem ipsum placeholder articles (real content deferred)

### Contact Page

- [x] **CONT-01**: Hero section with headline "Contact Dan" and subhead
- [x] **CONT-02**: Primary CTA button opens mailto link (Email Dan)
- [x] **CONT-03**: Copy Email button copies address to clipboard with confirmation feedback
- [x] **CONT-04**: LinkedIn message button links to Dan's LinkedIn profile
- [x] **CONT-05**: Microcopy displays "Typical reply: 1-2 business days" and urgent-subject tip
- [x] **CONT-06**: Form has inline validation for email format and message minimum length
- [x] **CONT-07**: Clear success state: "Sent -- thanks. I'll reply within X."
- [x] **CONT-08**: Failure state with direct email fallback
- [x] **CONT-09**: Loading state with disabled submit button
- [x] **CONT-10**: JS-disabled fallback (email-only fallback or basic HTML form post)
- [x] **CONT-11**: Honeypot field for spam protection (existing -- preserve)
- [x] **CONT-12**: Rate limiting by IP (existing -- preserve)
- [x] **CONT-13**: Server-side validation with length limits (existing -- preserve)
- [x] **CONT-14**: "Other Ways to Reach Me" section with LinkedIn and GitHub links
- [x] **CONT-15**: Privacy/retention disclosure
- [x] **CONT-16**: Analytics event stubs for copy, click, form start, submit, error
- [x] **CONT-17**: Works on mobile -- single-column layout with large tap targets
